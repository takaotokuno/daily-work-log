I learned the basic implementation of asynchronous communication processing using the fetch API in TypeScript.

## Bad Example

```ts
async function getUser(id: string) {
  const res = await fetch(`/api/users/${id}`);
  const data = await res.json();
  return data;
}
```

## Problems

- The `response.json()` method itself may fail.
- This function does not handle network errors.
- The `id` is embedded directly in the URL.
  The URL may break if the `id` contains characters such as `/` or `?`.
- There is no runtime validation of the response.

## Good Example

```ts
import { z } from "zod";

const UserSchema = z.object({
  id: z.string(),
  name: z.string(),
});

type User = z.infer<typeof UserSchema>;

async function getUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${encodeURIComponent(id)}`);

  if (!response.ok) {
    throw new Error(`Failed to fetch user: ${response.status}`);
  }

  const data: unknown = await response.json();
  return UserSchema.parse(data);
}
```
